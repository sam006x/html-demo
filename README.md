# Web Development Practice Repository

This repository contains different projects and experiments I have worked on while learning web development.

---

##  Contents

###  Login System

A PHP and MySQL-based authentication system with features like:

* User registration
* Login/logout
* Password reset
* Dashboard access
* Basic security features (e.g. brute force protection)

📂 Folder: `login-system/`

---

### 🌐 HTML Practice Pages

Basic frontend pages created while learning HTML and CSS.

Files include:

* `index.html`
* `Dashboard.html`
* `order.html`

These were used to practice layout, structure, and styling.

---

## 🛠️ Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## 🚀 How to Use

Each project is independent:

* For the **login system**, go into the `login-system/` folder and follow its README for setup instructions.
* The HTML files can be opened directly in a browser.
